What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a way for web browsers to represent the content of a webpage so that it can be manipulated using programming languages like JavaScript. It organizes the HTML elements (like paragraphs, images, links,
etc.) into a tree-like structure that programmers can interact with.

How do you select an element from the DOM?

To select an element from the DOM (Document Object Model) in JavaScript, you can use a variety of built-in methods. These methods allow you to grab specific elements or groups of elements from your webpage so that you can manipulate them. Common Ways to Select DOM Elements:

getElementById():

Selects a single element by its id attribute. IDs are unique, so this returns just one element

getElementsByClassName():

Selects all elements that have a specific class. Returns an HTMLCollection (a list of elements), so you might need to loop through it or access a specific item.

getElementsByTagName():

Selects all elements by their HTML tag (e.g., div, p, img). Also returns an HTMLCollection.

querySelector():

Selects the first element that matches a CSS selector. You can select by class, ID, tag, or even more complex CSS selectors.

querySelectorAll():

Selects all elements that match a CSS selector. Returns a NodeList (similar to an array), so you can loop through it or access individual elements.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JavaScript where a single event listener is attached to a parent element rather than adding multiple event listeners to individual child elements. This is especially useful in the context of the DOM when you have many similar child elements or dynamically created elements.

How do you manipulate an element's attributes and styles using the DOM?

In the DOM, you can easily manipulate an element"s attributes (such as id, src, href) and styles (like color, font-size, background) using JavaScript. This allows you to dynamically change the content, appearance, and behavior of elements on the page. 1. Manipulating Attributes You can get, set, or remove HTML attributes on elements using JavaScript. Common methods: getAttribute(attribute): Retrieves the value of a specified attribute. setAttribute(attribute, value): Sets or changes the value of a specified attribute. removeAttribute(attribute): Removes a specified attribute from an element.
